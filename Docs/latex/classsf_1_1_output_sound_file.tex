\doxysubsection{Opis szczegółowy}
Provide write access to sound files. 

This class encodes audio samples to a sound file. It is used internally by higher-\/level classes such as \doxylink{classsf_1_1_sound_buffer}{sf\+::\+Sound\+Buffer}, but can also be useful if you want to create audio files from custom data sources, like generated audio samples.

Usage example\+:

\DoxyCodeLine{\textcolor{keywordflow}{if}\ (!file.\mbox{\hyperlink{classsf_1_1_output_sound_file_ae5e55f01c53c1422c44eaed2eed67fce}{openFromFile}}(\textcolor{stringliteral}{"{}music.ogg"{}},\ 44100,\ 2))}
