\doxysubsection{Opis szczegółowy}
Klasa statyczna odpowiedzialna za wczytywanie poziomów. 

Implementuje parsowanie plików graficznych (PNG) na strukturę mapy kafelkowej. Mapuje kolory pikseli na identyfikatory kafelków (Tile ID).
