\doxysubsection{Opis szczegółowy}
Klasa obsługująca sterowanie (mechanika Drag \& Shoot). 

Odpowiada za wykrywanie kliknięć na piłce, przeciąganie myszką (celowanie) oraz obliczanie wektora siły uderzenia.
