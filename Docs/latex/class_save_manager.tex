\doxysubsection{Opis szczegółowy}
Klasa statyczna obsługująca trwałość danych gry (system zapisu/odczytu). 

Zarządza odczytem i zapisem postępów gracza (odblokowane poziomy, najlepsze wyniki) do lokalnego pliku tekstowego.
