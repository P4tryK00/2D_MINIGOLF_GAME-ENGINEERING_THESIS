\doxysubsection{Opis szczegółowy}
Utility class that measures the elapsed time. 

\doxylink{classsf_1_1_clock}{sf\+::\+Clock} is a lightweight class for measuring time.

Its provides the most precise time that the underlying OS can achieve (generally microseconds or nanoseconds). It also ensures monotonicity, which means that the returned time can never go backward, even if the system time is changed.

Usage example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classsf_1_1_clock}{sf::Clock}}\ clock;}
\DoxyCodeLine{...}
\DoxyCodeLine{Time\ time1\ =\ clock.\mbox{\hyperlink{classsf_1_1_clock_abe889b42a65bcd8eefc16419645d08a7}{getElapsedTime}}();}
\DoxyCodeLine{...}
\DoxyCodeLine{\mbox{\hyperlink{classsf_1_1_time_acba0cfbc49e3a09a22a8e079eb67a05c}{Time}}\ time2\ =\ clock.\mbox{\hyperlink{classsf_1_1_clock_a123e2627f2943e5ecaa1db0c7df3231b}{restart}}();}

\end{DoxyCode}


The \doxylink{classsf_1_1_time}{sf\+::\+Time} value returned by the clock can then be converted to a number of seconds, milliseconds or even microseconds.
