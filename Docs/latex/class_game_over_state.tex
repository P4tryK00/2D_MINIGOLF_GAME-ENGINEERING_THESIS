\doxysubsection{Opis szczegółowy}
Stan gry wyświetlany po przegranej (\doxylink{class_game}{Game} Over). 

Klasa dziedzicząca po \doxylink{class_state}{State}. Wyświetla komunikat o końcu gry, przyciemnia tło i pozwala na restart poziomu lub powrót do menu.
