Provide a string representation for a given scancode.   

The returned string is a short, non-\/technical description of the key represented with the given scancode. Most effectively used in user interfaces, as the description for the key takes the users keyboard layout into consideration.

\begin{DoxyWarning}{Ostrzeżenie}
The result is OS-\/dependent\+: for example, \doxylink{structsf_1_1_keyboard_1_1_scan_aa42fbf6954d6f81f7606e566c7abe875a5e1b12c4476475396c6f04eccbbf04f7}{sf\+::\+Keyboard\+::\+Scan\+::\+LSystem} is "{}\+Left Meta"{} on Linux, "{}\+Left Windows"{} on Windows and "{}\+Left Command"{} on mac\+OS.
\end{DoxyWarning}
The current keyboard layout set by the operating system is used to interpret the scancode\+: for example, \doxylink{classsf_1_1_keyboard_acb4cacd7cc5802dec45724cf3314a142ab50635b9c913837d1bd4453eec7cb506}{sf\+::\+Keyboard\+::\+Semicolon} is mapped to "{};"{} for layout and to "{}é"{} for others.

\begin{DoxyReturn}{Zwraca}
The localized description of the code   
\end{DoxyReturn}
