\doxysubsection{Opis szczegółowy}
Implementation of input stream based on a file. 

This class is a specialization of \doxylink{classsf_1_1_input_stream}{Input\+Stream} that reads from a file on disk.

It wraps a file in the common \doxylink{classsf_1_1_input_stream}{Input\+Stream} interface and therefore allows to use generic classes or functions that accept such a stream, with a file on disk as the data source.

In addition to the virtual functions inherited from \doxylink{classsf_1_1_input_stream}{Input\+Stream}, \doxylink{classsf_1_1_file_input_stream}{File\+Input\+Stream} adds a function to specify the file to open.

SFML resource classes can usually be loaded directly from a filename, so this class shouldn\textquotesingle{}t be useful to you unless you create your own algorithms that operate on an \doxylink{classsf_1_1_input_stream}{Input\+Stream}.
