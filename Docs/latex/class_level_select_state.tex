\doxysubsection{Opis szczegółowy}
Stan wyboru poziomu. 

Wyświetla listę dostępnych poziomów. Komunikuje się z Save\+Managerem, aby zablokować poziomy, do których gracz nie ma jeszcze dostępu (oznaczone jako LOCKED).
