\doxysubsection{Opis szczegółowy}
Statyczny menedżer zasobów dla tekstur SFML. 

Implementuje wzorzec Pyłek (Flyweight) poprzez przechowywanie załadowanych tekstur w mapie. Zapobiega to wielokrotnemu ładowaniu tego samego zasobu z dysku, oszczędzając pamięć.
