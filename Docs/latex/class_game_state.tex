\doxysubsection{Opis szczegółowy}
Główny stan rozgrywki. 

Klasa ta implementuje pętlę gry, w tym\+:
\begin{DoxyItemize}
\item Fizykę ruchu piłki (tarcie, odbicia, grawitacja przy skokach).
\item Detekcję kolizji z mapą kafelkową (\doxylink{class_tile_map}{Tile\+Map}).
\item Logikę interakcji z elementami specjalnymi (woda, piasek, rampy, dołek).
\item Zarządzanie cyklem życia (śmierć, wygrana, pauza). 
\end{DoxyItemize}
