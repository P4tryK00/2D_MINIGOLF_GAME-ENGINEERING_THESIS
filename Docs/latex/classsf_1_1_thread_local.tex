\doxysubsection{Opis szczegółowy}
Defines variables with thread-\/local storage. 

This class manipulates void\texorpdfstring{$\ast$}{*} parameters and thus is not appropriate for strongly-\/typed variables. You should rather use the \doxylink{classsf_1_1_thread_local_ptr}{sf\+::\+Thread\+Local\+Ptr} template class.
