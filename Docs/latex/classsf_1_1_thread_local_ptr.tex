Pointer to a thread-\/local variable. 

\doxylink{classsf_1_1_thread_local_ptr}{sf\+::\+Thread\+Local\+Ptr} is a type-\/safe wrapper for storing pointers to thread-\/local variables. A thread-\/local variable holds a different value for each different thread, unlike normal variables that are shared.

Its usage is completely transparent, so that it is similar to manipulating the raw pointer directly (like any smart pointer).
