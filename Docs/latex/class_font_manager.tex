\doxysubsection{Opis szczegółowy}
Klasa statyczna zarządzająca zasobami czcionek w grze. 

Odpowiada za ładowanie, przechowywanie i udostępnianie obiektów sf\+::\+Font. Zapobiega wielokrotnemu ładowaniu tych samych plików z dysku.
