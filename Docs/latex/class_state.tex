\doxysubsection{Opis szczegółowy}
Abstrakcyjna klasa bazowa reprezentująca pojedynczy stan gry (np. Menu, Gra, Pauza). 

Klasa ta definiuje interfejs, który muszą zaimplementować wszystkie konkretne stany gry. Wykorzystuje wzorzec projektowy "{}\+Stan"{} (\doxylink{class_state}{State} Pattern), pozwalając State\+Managerowi na jednolite zarządzanie różnymi etapami rozgrywki.
