\doxysubsection{Opis szczegółowy}
Stan gry wyświetlany po pomyślnym ukończeniu poziomu. 

Klasa ta odpowiada za\+:
\begin{DoxyItemize}
\item Wyświetlenie podsumowania poziomu (wynik/ilość uderzeń).
\item Zapisanie postępów gracza (odblokowanie kolejnego poziomu).
\item Nawigację do następnego poziomu, powtórzenia obecnego lub powrotu do menu. 
\end{DoxyItemize}
