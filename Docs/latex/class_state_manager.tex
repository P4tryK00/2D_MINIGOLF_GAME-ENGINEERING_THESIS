\doxysubsection{Opis szczegółowy}
Zarządza stosem stanów gry. 

Klasa odpowiedzialna za dodawanie, usuwanie i przełączanie między różnymi stanami gry. Obsługuje przejścia między stanami w bezpiecznym momencie (na początku pętli gry), aby uniknąć błędów pamięci podczas aktualizacji.
